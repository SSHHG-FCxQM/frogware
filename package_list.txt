You must be using Python 3.8.
Run the following command in Anaconda command prompt after installing
Anaconda or Miniconda:

conda create --name frogenv python=3.8 numpy scipy pyqtgraph pandas

conda activate frogenv
pip install pyusb=1.0.0b1

Then download the thorlabs_apt package as a zip file from its github page:
https://github.com/qpit/thorlabs_apt
cd into its directory then run:

conda activate frogenv
python setup.py install

Then install the Thorlabs Apt software from thorlabs:
https://www.thorlabs.com/software_pages/ViewSoftwarePage.cfm?Code=Motion_Control&viewtab=1
Locate the Apt software installation directory in C:\Programs and copy the Apt.dll
file into the same folder as the thorlabs_apt package installation folder.
You should be able to find the thorlabs_apt package installation folder under:
C:\Users\username\Miniconda3\envs\frogenv\Lib\site-packages\
